## Présentation :
Cette application est une **Single Page Application (SPA)** développée en JavaScript pur (Vanilla JS) permettant de gérer une base de données de personnages de jeux vidéo ou de jeux de rôle.

## Installation et Lancement :

1. **Cloner le dépôt** :
    ```bash
    git clone https://github.com/Emrecan45/JS_TP_note.git
    ```
2. **Lancer le serveur de données** (nécessite `json-server`) :
    ```bash
    npx json-server data/db.json --port 3000
    ```
3. **Lancer le site** :
    ```bash
    npx serve
    ```
    
    Si le port 3000 est deja utilisé :
    ```bash
    npx kill-port 3000
    ```
